Фотографии при наведении курсора становятся более темными.

Это можно сделать так:

html:

<figure>
  <img src="img/page1-img1.jpg" alt="">
  <figcaption></figcaption>
</figure>

css:

figure {
  width: calc((100% - 30px) / 3);
  margin-left: 15px;
  position: relative;
}

img {
  width: 100%;
}

figcaption {
  width: 100%;
  height: 100%;
  position: absolute;
  left: 0;
  top: 0;
}
figure:hover figcaption {
  background: rgba(0, 0, 0, 0.3); 
}
